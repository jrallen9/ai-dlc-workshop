# Job Application Website with CDK

Build a job application management site using AWS CDK and AI-DLC methodology

## Purpose

Duration: ~3 hours | Project: Job Application Site | Stack: React / Vite / AWS CDK

## Source

- Reconstructed page: `../../website/public/job-application-cdk.html`
- Original source details: `../../source-mirror/manifests/crawl-manifest.json`

## Prerequisites

- Basic React/JavaScript knowledge
- Access to an AI assistant (Kiro, Claude, Cursor, etc.)
- AWS account with CDK bootstrapped
- Node.js and npm installed
- Code editor

## How To Run Or View

Open `../../website/public/job-application-cdk.html` in a browser, or serve `website/public/` with the Java static server once Phase 5 is complete.
