# Task Manager Application

Build a Simple Task Manager - Learn AI-DLC by building something real you can use

## Purpose

Duration: ~2.5 hours | Project: Personal Task Manager | Stack: Python / Flask

## Source

- Reconstructed page: `../../website/public/task-manager.html`
- Original source details: `../../source-mirror/manifests/crawl-manifest.json`

## Prerequisites

- Basic Python knowledge
- Access to an AI assistant (Kiro, Claude, Cursor, etc.)
- AWS account (free tier sufficient)
- Code editor

## How To Run Or View

Open `../../website/public/task-manager.html` in a browser, or serve `website/public/` with the Java static server once Phase 5 is complete.
