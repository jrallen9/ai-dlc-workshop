# E-Commerce Product Catalog

Duration: 2-3 hours | Build a product catalog system using AI-DLC methodology

## Purpose

Apply AI-DLC methodology to build an e-commerce product catalog. Customers can browse products, search, view details, and add items to cart. Admins can manage products and inventory.

## Source

- Reconstructed page: `../../website/public/ecommerce-catalog.html`
- Original source details: `../../source-mirror/manifests/crawl-manifest.json`

## Prerequisites

- Basic programming knowledge
- AWS account (free tier sufficient)
- Access to AI assistant (Kiro, Claude, Cursor, etc.)
- Code editor

## How To Run Or View

Open `../../website/public/ecommerce-catalog.html` in a browser, or serve `website/public/` with the Java static server once Phase 5 is complete.
