# S3 File-Sharing System

Build a complete web-based file sharing system backed by S3

## Purpose

Duration: ~4 hours | Project: S3 File-Sharing System

## Source

- Reconstructed page: `../../website/public/s3-file-sharing.html`
- Original source details: `../../source-mirror/manifests/crawl-manifest.json`

## Prerequisites

- Basic React/JavaScript and Python knowledge
- Access to an AI assistant (Kiro, Claude, Cursor, etc.)
- AWS account with CLI configured
- Node.js, npm, Python, and AWS SAM CLI installed
- Code editor

## How To Run Or View

Open `../../website/public/s3-file-sharing.html` in a browser, or serve `website/public/` with the Java static server once Phase 5 is complete.
